## 詳細測試案例

### TC-001：驗證 AI 分頁連線偵測

**類別**：連線管理  
**優先級**：P0  
**前置條件**：擴充功能已載入

**測試步驟**：
1. 開啟擴充功能 Side Panel
2. 開啟 Claude 網站 (https://claude.ai) 並登入
3. 開啟 ChatGPT 網站 (https://chatgpt.com) 並登入
4. 開啟 Gemini 網站 (https://gemini.google.com) 並登入
5. 觀察 Side Panel 中各 AI 的連線狀態

**預期結果**：
- [ ] Claude 狀態顯示「已連線」（綠色）
- [ ] ChatGPT 狀態顯示「已連線」（綠色）
- [ ] Gemini 狀態顯示「已連線」（綠色）

---

### TC-002：驗證分頁關閉後狀態更新

**類別**：連線管理  
**優先級**：P0  
**前置條件**：三個 AI 分頁皆已開啟且已連線

**測試步驟**：

- [ ] 日誌顯示傳送失敗訊息
- [ ] Claude 狀態更新為「未找到」（灰色）

以下腳本可在 Side Panel 的 DevTools Console 中執行，用於半自動化測試訊息解析邏輯：

```javascript
// ===== 測試 parseMessage 函式 =====

// 測試 /mutual 指令
console.log('=== /mutual 指令測試 ===');
console.log('Case 1:', parseMessage('/mutual'));
console.log('Case 2:', parseMessage('/mutual 自訂提示'));

// 測試 /cross 指令
console.log('=== /cross 指令測試 ===');
console.log('Case 1:', parseMessage('/cross @Claude <- @ChatGPT 評價一下'));
console.log('Case 2:', parseMessage('/cross @Claude @Gemini <- @ChatGPT'));

// 測試 @ 提及語法
console.log('=== @ 提及語法測試 ===');
console.log('Case 1:', parseMessage('@Claude 評價一下 @ChatGPT'));
console.log('Case 2:', parseMessage('@ChatGPT 怎麼看 @Claude'));

// 測試一般訊息
console.log('=== 一般訊息測試 ===');

console.log('Case 2:', parseMessage(''));
```
